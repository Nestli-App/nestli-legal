# Nestli

Nestli is a private Android app for shared household management.

This repository hosts:

- The marketing and legal site served at **[nestliapp.com](https://nestliapp.com)** (Cloudflare Workers, static).
- The **Global Android APK** distributed via [GitHub Releases](https://github.com/jimmy1992abc/nestli-legal/releases) for users who cannot, or prefer not to, install via Google Play.

> The Nestli application source code is **not** included in this repository.

## Download

| Audience | Channel |
| --- | --- |
| Most Android users | Google Play Store *(recommended)* |
| Non-Play-store global users | [Latest GitHub Release](https://github.com/jimmy1992abc/nestli-legal/releases/latest) |
| 中国大陆地区用户 (中文版) | [Gitee Mirror](https://gitee.com/jimmy1992abc/nestli) |
| iOS users | App Store *(coming soon)* |

The APK published here is the same signed build released to Google Play.

**Install:** download `nestli-release.apk`, then on Android allow "Install unknown apps" for your browser (Settings → Apps → Special app access) and open the file. Full instructions: <https://nestliapp.com/downloads/>.

## Regional availability

Some features (AI assistance, paid plans, voice notes, in-app purchases, marketing) are only available in supported regions. The China-specific build hosted on Gitee additionally disables these features for compliance — see the [cross-border transfer notice](https://nestliapp.com/cross-border-transfer/).
